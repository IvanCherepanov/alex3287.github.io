CRAAP

Тест CRAAP - это тест, разработанный Сарой Блейксли и ее командой библиотекарей из Университета штата Калифорния в Чико (CSU Chico) для проверки достоверности источников по всем академическим дисциплинам.
Из-за огромного количества источников, существующих в Интернете, может быть трудно определить, насколько эти источники заслуживают доверия для использования в качестве инструментов для исследований.
Таким образом, тест CRAAP позволяет преподавателям и студентам определить, можно ли доверять их источникам.
